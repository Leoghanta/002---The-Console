The Console

Workshop 1 is all about getting you started with the console. We will be looking at the Developer Tools and understanding how to look at the code behind web pages. We will look at the back end console and using JavaScript code to make things happen.

Git Hub

Before we get started, all of your course notes are available for you to download as a web page using Git Hub. Git Hub is a repository that allows online access, multi-user support, version control, basically everything that you need for a software development business. Git Hub has a free version, and also a student version. It is recommended you use your personal e-mail address to sign up to Git Hub so your repositories will stay with you long after your college account has been shut down.

The first URL for this repository is JS002.The Console and you have a couple of options to grab this repository.
